🚀 About the Project

This project presents a Smart Drainage and Flood Monitoring System designed for Madurai city. It integrates GIS mapping, real-time weather data, and IoT-based drainage sensors to monitor and predict flood risks.

The system identifies clogged drainage points and displays them on an interactive map. Users can report issues with live location and images, enabling faster response and maintenance.

A predictive model analyzes rainfall and drainage conditions to estimate flood risk levels, helping authorities take preventive action.
